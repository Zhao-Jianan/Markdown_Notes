# 左侧工具栏
**填充层不能使用这些工具，绘制层和遮罩可以使用**
## 画笔（快捷键1）
### 画笔属性
- 跟随路径------------------------开启后，画笔图案的方向会随绘制的路径而改变
- 大小抖动------------------------让画笔绘制大小有一定随机
- 流量抖动------------------------让画笔绘制的透明度有一定随机
- 角度抖动------------------------让画笔绘制的图案旋转有一定随机
- 校准----------------------------控制画笔绘制的方向
**视图中长按Ctrl键，会弹出画笔相关的快捷键**
- Ctrl+左键左右拖动控制画笔流量
- Ctrl+左键上下拖动控制画笔旋转
- Ctrl+右键左右拖动控制画笔大小
- Ctrl+右键上下拖动控制画笔硬度
- Shift+左键绘制直线

### Alpha透贴
- Alpha透贴-----------------------将画笔图案拖入

### 模板
- 模板映射------------------------把图案拖到模板映射中，会在屏幕中出现一个屏幕蒙版，绘制时屏幕蒙版内的区域可以被绘制，屏幕蒙版外的区域不能被绘制
**视图中按S键，会弹出模板相关的快捷键**

### 材质
- Color---------------------------颜色通道
- Height--------------------------高度通道
- Rough---------------------------粗糙度通道
- Metal---------------------------金属度通道
- Nrm-----------------------------法线通道
- 材质模式------------------------可以将内置的材质拖入，再用画笔绘制

## 橡皮擦（快捷键2）

## 投射笔刷（快捷键3）
和模板映射类似，不同的是投射笔刷投射的是可以有颜色通道的贴图
**使用时点击投射笔刷，或按快捷键3，然后选择一张纹理贴图，拖入材质通道中，拖入颜色通道就控制颜色，拖入高度通道就控制高度，然后用画笔在模型表面绘制**
**视图中按S键，会弹出模板相关的快捷键**

## 几何体填充（快捷键4）
一般配合图层遮罩使用，用来编辑图层遮罩
- 三角形填充----------------------选择三角面
- 几何体填充----------------------选择四边面
- 模型填充------------------------选择模型
- UV块填充------------------------在UV视图中选择整个UV块

## 涂抹工具（快捷键5）
让两个图层混合的更好，使用时添加一个绘制层，放在两个图层的上面，叠加模式改为Passthough/穿透

## 图章工具（快捷键6）
与PS中图章工具相同，选择图章源时按V+鼠标左键

## 材质选择器
类似于PS中的吸管工具，可以吸取材质并赋予给画笔，必须在绘制层中使用

# Texture Set Settings/纹理集设置
## 烘焙模型贴图
用来将高模信息烘焙到低模上
### 通用参数
  - 输出尺寸------------------------设置烘焙的几个通道贴图的尺寸，最好与项目贴图尺寸设置相同
  - 扩张宽度------------------------UV边界向外扩张几像素，用来规避UV缩放出现的错误，一般设置为1或2
  - 应用Diffusion漫反射-------------帮助扩张宽度进行计算，一般需要勾选
  - 高模----------------------------导入高模
  - 使用包裹------------------------勾选后会在投射光线时增加一个限定盒子，映射的范围不会超过盒子范围
  - 包裹文件------------------------导入一个文件作为包裹盒子，一般不导入，默认的包裹就可以
  - 最大前方距离--------------------低模表面到包裹盒子的最大距离，这个距离决定了能烘焙出多少细节
  - 最大后方距离--------------------低模内部投射距离，因为有的高模不一定完全能包裹住低模，可能有的部分在低模内部，这个距离控制低模内部投射
  - 相对于边界框--------------------一般保持勾选，让高模不超过边界框
  - 平均法线------------------------一般保持勾选，计算法线时让表面平滑
  - 忽略背面------------------------一般保持勾选，会在烘焙时模型薄的地方不会出现穿透
  - 匹配
    - 总是----------------------------一般选择总是自动匹配
    - 按照模型名称--------------------如果模型有很多组件，且互相离得很近，此时使用总是自动匹配容易出错，要使用按照模型名称匹配
  - 抗锯齿--------------------------抗锯齿级别越高，烘焙质量越好，对电脑性能要求也越高
  - High poly mesh suffix-----------高模文件名匹配，如果烘焙的模型有多个，设置一个高模的文件名后缀，进行自动匹配
  - Low poly mesh suffix------------低模文件名匹配，如果烘焙的模型有多个，设置一个低模的文件名后缀，进行自动匹配
